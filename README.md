Para rodar o frontend:
- cd /web
- yarn
- yarn run dev

Para rodar o backend
- Abra o projeto no Visual Studio
- Altere a variável "DefaultConnection" no arquivo appsettings.json para a sua conexão de banco de dados
- Rode o projeto
